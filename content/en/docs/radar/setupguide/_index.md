The setup guide will include these steps:

1. Understanding which hardware to use
2. Downloading ORCA on your laptop
3. Getting your Raspberry Pi set up
4. Getting the ORCA code on the Raspberry Pi
5. Connecting to the SDR with your Raspberry Pi
6. Testing
    - Transmitter test using a spectrum analyzer
    - Receiver test using a wave and signal generator
    - Loopback test
    - Outdoor test

At the end of this process, your SDR and Raspberry Pi should be able to be used as a radar.
